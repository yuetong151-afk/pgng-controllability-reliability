# Test-Retest Reliability of a Pavlovian Go/No-Go Task with a Controllability Manipulation

Analysis code for BSc dissertation, King's College London (2026).

## Study Overview
This study examined how environmental controllability modulates 
Pavlovian-instrumental arbitration, and evaluated the test-retest 
reliability of behavioural and computational measures extracted 
from a modified Pavlovian Go/No-Go (PGNG) task.

## Notebooks
- `01_Demographic.ipynb`: Demographic characteristics and task appraisals
- `02_Descriptive.ipynb`: Behavioural analyses and model-free reliability
- `03_Modeling.ipynb`: Computational modelling and parameter reliability
